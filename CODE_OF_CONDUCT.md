# Code of Conduct

## Our Pledge
We aim to create a welcoming, professional environment for contributors to the **CCM ESP32 Vision Node** and CCM Code projects.

## Our Standards
Positive behaviors:
- Clear, constructive communication  
- Respectful collaboration  
- Thoughtful code reviews  
- Helpful and actionable feedback  

Unacceptable behaviors:
- Harassment or discrimination  
- Personal attacks or insults  
- Dismissing contributions without explanation  
- Releasing private information  

## Responsibilities of Maintainers
- Define project standards  
- Review contributions fairly  
- Enforce behavioral expectations  
- Apply corrective action when needed  

## Scope
This Code of Conduct applies:
- Throughout issues, PRs, and discussions  
- In any public interactions representing the project
